## Installation

You can install the package via composer:

```bash
composer require khamdullaevuz/laravel-slug
```

## Usage

```php
use KhamdullaevUz\LaravelSlug\LaravelSlug;

LaravelSlug::make('Привет, мир'); // privet-mir
```
